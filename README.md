# Rihanna

Rihanna is a database-backed job distributed job queue for Elixir.


## Installation

Add `rihanna` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:rihanna, "~> 0.1.0"}
  ]
end
```
